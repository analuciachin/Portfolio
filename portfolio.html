Arcade Game

Arcade Game was part of the Front-end Development Nanodegree program.

The user should choose one out of 5 caracters to start the game. The game consists in getting the collectables and reaching the water.

Every time the enemy hits you, you lose 1 point in the score. You can not move to a position if there is a rock.

Neighborhood Map

Neighborhood Map was part of the Front-end Development Nanodegree program.

The goal of this project was to develop a single page application featuring a map of a neighborhood using React.

Neighborhood Map shows the Montreal city featuring some turistic locations. A third-party data (Wikipedia API) gives us more details about each of these locations. The map also allows the user to browse the content in different ways.

Memory Game

Memory game was part of the Front-end Development Nanodegree program.

The game starts once the page is loaded and the timer starts. At the beginning of the game, all cards are flipped over (no images are shown) and the player has to open two cards (one card at a time). If the image on both cards match, the cards will shake and turn to green color, otherwise, the cards will flip over and hide the image. The game ends when the player finds all the matches.

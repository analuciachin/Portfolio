Neighborhood Map

The Neighborhood Map was the last project from the Front-end development nanodegree curriculum at Udacity. The project had to be done in React and we had to build it from scratch. We were not given any initial code to work on, only the guidelines on how to complete the project.

The goal of this project was to develop a single page application featuring a map of a neighborhood.

Neighborhood Map shows the Montreal city featuring some turistic locations. A third-party data (Wikipedia API) gives us more details about each of these locations. The map also allows the user to browse the content in different ways.

Once the map is loaded, the user will see a list of the touristic places in Montreal on the left and a map of Montreal with the red location pins on the right side.

When a location pin is clicked, it will change its color to blue and some information regarding that location will show up with a link to the full Wikipedia article.

The user can also search a location by typing its name in the search box above the location list. It will remove all other location pins but the one searched by the user.

Another way to browse through the content it is by clicking at an item in the location list. Clicking a location in the list will change its pin color to blue and show the information regarding that location.

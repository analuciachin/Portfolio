My Movies

After completing the Front-end development nanodegree from Udacity and the React course from TylerMcginnis.com. I have decided to create a project that was unique and where I could practice the React knowledge I have gained in those courses. That is how My Movies project was born.

I have created a login page to learn how to simulate an API call using promises.

username: abc / password: 123

After a successful login, the user is taken to the search page. In this page, the user can enter the period of the movie review.

If there is no reviews in period, a message will be displayed to inform the user.

After the search is completed, up to 20 movies picked by the critics (according to NYT API) will be loaded to the 'All' tab.
